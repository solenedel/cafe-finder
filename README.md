# ☕ Café finder

Find cafés near you, with ratings from other users. Save your favourite cafés to a list for quick reference. 
 
## Dependencies
- Styled components
- React router
- React google maps **(not google maps react)**

## Running the app
- Run in development mode: `npm start`
